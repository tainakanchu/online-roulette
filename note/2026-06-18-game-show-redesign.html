🎯 作業ノート: Online Roulette「Game Show」UI リデザイン

開始日: 2026-06-18 / ブランチ: feat/game-show-redesign / worktree: roulette-feat-game-show-redesign

タスク概要・目的

ユーザーが Online roulette repository.zip として配布したデザインハンドオフ（design_handoff_roulette_redesign/）に基づき、 既存のライト基調・カード型 UI を、ネオンの「ゲームショー」ステージ風 UI に一新する。

  • ダークな紫/ピンクのグラデ背景 + 浮遊する光の blob
  • glassmorphism パネル、電球付きの回転ホイール + 中央ハブ
  • textarea の代わりに チップ式の選択肢エディタ（bulk edit は残す）
  • 勝利時の 紙吹雪（confetti） + 大きな当選演出
  • より緊張感のある Battle モード（カウントダウン、リーダーバナー、メダル/王冠、ゆっくりサドンデス）
  • Fredoka フォント、5言語 i18n は維持

Issue / 指示の自分の理解

  • ハンドオフ HTML（.dc.html）はそのまま出荷しない。既存の monorepo 構造（React18 + TS + Vite、packages/coreのロジック）に合わせて再現する。
  • 既存のコンポーネント/フック構造を再利用し、見た目と新しいインタラクション層のみを追加・変更する。
  • ユーザー指示「既存の構造に合わせていい感じに。終わったらPR出して」。
  • ★ユーザー判断: ポインタは3時（右）の位置を維持。→ プロトタイプの「上ポインタ + 選択計算変更（270度式）」は採用せず、packages/core の選択ロジックとテストは無変更。矢印だけネオン調（ゴールド）にして右側に描画する。

既存コードの構造（調査結果）

領域ファイルメモ
ロジック(core)colors / roulette / grouping / battle / shuffleCOLORS, getColorBrightness, divideIntoGroups, drawBattleSequence, createDefaultRouletteLogic 等をエクスポート。無変更で再利用。
状態フックuseRouletteOptions / useRouletteAnimation / useRouletteHistory / useGroupDivision / useBattleMode / useSnackbarこれらを温存しつつ拡張する。
CSSplain global CSS（CSS Modulesではない）を import 副作用で読み込みハンドオフは「CSS Modules」と書くが実体は global class CSS。既存方式に合わせ class ベースで全面リスタイル。

採用方針 / 設計判断

  • confetti: useConfetti フックを新設（canvas ref + fire()）。パネル内に absolute 配置。roulette/groups/battle の勝利時に App 側で発火。
  • 選択肢エディタ: OptionEditor を新設（chips + 追加input + bulk textarea + shuffle）。useRouletteOptions に newOption / addOption / removeOption / clearOptions / showBulkEdit を追加。
  • 選択肢変更時の stale 結果リセット: App 側で add/remove/clear 時に roulette結果と groups をリセット。battle は optionsKey effect で自動リセット。
  • ホイール: RouletteCanvas を 460×460 固定 + 電球/ハブ/右側ゴールド矢印に。クリックで spin。選択計算は core 既存式のまま。
  • コピー/DL: プロトタイプ同様、合成画像ではなくホイール canvas を直接書き出す方向で RouletteActions を簡素化（新しい暗色ホイールが白背景合成と相性悪いため）。
  • Battle: useBattleMode に countdownValue を追加し、start→カウントダウン(3/2/1/GO)→レース。サドンデスは初回800ms/以降650ms + 勝者確定前に1200ms hold。リーダーバナー/メダル/王冠を BattleBars・BattleMode に。
  • テーマCSS: 旧 light/dark の prefers-color-scheme 切替を廃し、常時ネオンダークに。index.css のグローバル button/色スキームを中和。
  • i18n: 新キー追加（ui.tagline / ui.empty / ui.add / ui.bulk / ui.clear / ui.leading / ui.go）を全5言語に。既存の title/spin 等の文言はプロトタイプ寄りに微調整（spin="SPIN"/"回す" 等は維持）。
